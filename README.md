# House Price Prediction using Machine Learning

## Overview
This notebook demonstrates a beginner-friendly machine learning project for predicting house prices using regression techniques. The project walks through the complete ML workflow, including data preprocessing, exploratory data analysis, feature engineering, model training, and evaluation.

---

## What This Project Covers
- Loading and understanding housing data  
- Handling missing values and cleaning data  
- Exploratory Data Analysis (EDA)  
- Encoding categorical variables  
- Splitting data into training and testing sets  
- Training a Linear Regression model  
- Evaluating performance using regression metrics  
- Saving the trained model for future use  

---

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn

This project uses a housing dataset containing various property-related features such as:
